    • The "code" package is used strictly to generate a new 7-digit code to be used for verification purposes. The code will be put into a user's "code" column in the database. Should a user enter a code that matches their auto-generated code, they will become "verified", allowing them to traverse through the app knowing that no bots will harm them. This code will be emailed to the user immediately after they sign up for CyFire by simply selecting, "Send Code".
